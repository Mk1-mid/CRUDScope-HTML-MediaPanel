Consejos para crear tareas

Sigue estos pasos para registrar una tarea efectiva:

  1. Escribe un título claro y descriptivo
  2. Detalla la descripción con información relevante
  3. Establece una fecha límite realista
  4. Asigna la prioridad según la urgencia

Niveles de prioridad disponibles:

  • 1-2: Baja prioridad
  • 3: Prioridad media
  • 4-5: Alta prioridad

Acerca del Proyecto

CRUDScope - HTML MediaPanel es un proyecto educativo diseñado para practicar el uso de etiquetas HTML semánticas, no semánticas y multimedia en el contexto de un sistema de gestión de tareas.

Este proyecto fue desarrollado como parte de una actividad individual enfocada en demostrar el dominio de HTML5 y sus diversas etiquetas, incluyendo aquellas menos comunes pero igualmente importantes para la estructura y accesibilidad web.

Para más información sobre el desarrollo de este proyecto, visita el repositorio en GitHub.
